# 필터 총정리 — 정의 · 동작원리 · 사용용도

> 상태 추정(칼만 계열)부터 전통 신호처리·적응 필터까지 한 장으로.
> 흐름: **큰 그림 → 갈래별 정의/원리/용도 → 선택 가이드 → 한 줄 요약**

---

## Part 0. 큰 그림 — "필터"는 두 세계가 섞여 있다

"필터"라는 말은 사실 **목적이 다른 두 계열**을 한 단어로 부르는 것이다.

| 계열 | 푸는 문제 | 대표 |
|------|-----------|------|
| **상태 추정 (베이지안)** | 노이즈 측정에서 숨은 *상태*(위치·속도)를 추정 | KF, EKF, UKF, Particle |
| **신호 처리 (주파수)** | 신호에서 원하는 *주파수 성분*만 통과/제거 | Butterworth, Median, EMA |

> 핵심 질문: **"동역학 모델이 있나?"**
> 있다 → 상태 추정(칼만 계열). 없다·단순 잡음만 → 신호처리 필터.

---

## Part A. 상태 추정 (베이지안) 필터

가정을 **선형→비선형→비가우시안** 순으로 완화하며 발전한다.

### A-1. 칼만 필터 (KF)
- **정의**: 선형 시스템 + 가우시안 노이즈에서 숨은 상태를 추정하는 **최적 추정기**.
- **동작원리**: `예측(모델로 다음 상태)` → `보정(측정으로 수정)` 2단계 반복.
  ```
  [예측]  x⁻ = F·x + B·u ,  P⁻ = F·P·Fᵀ + Q
  [보정]  K  = P⁻Hᵀ(HP⁻Hᵀ + R)⁻¹        # 칼만 게인
          x  = x⁻ + K(z − H·x⁻)
          P  = (I − K·H)·P⁻
  ```
  칼만 게인 `K`가 예측·측정의 불확실성(`P` vs `R`)을 비교해 **더 믿을 쪽에 가중**.
- **사용용도**: GPS 추적, 센서 융합, 항법, 경제 시계열.

### A-2. 확장 칼만 필터 (EKF)
- **정의**: 비선형 시스템용 칼만 필터.
- **동작원리**: 비선형 함수를 현재 추정점에서 **자코비안(1차 테일러)** 으로 선형화 후 KF 적용.
- **사용용도**: 로봇 SLAM, GPS/IMU 융합, 자율주행 위치추정. (널리 쓰이나 강한 비선형엔 발산 위험)

### A-3. 무향 칼만 필터 (UKF)
- **정의**: 미분 없이 비선형을 다루는 칼만 필터.
- **동작원리**: 분포 대표점인 **시그마 포인트** 몇 개를 비선형 함수에 직접 통과시킨 뒤 평균·공분산 복원(Unscented Transform).
- **사용용도**: 자코비안 계산이 어렵거나 비선형이 강한 항법·자세 추정.

### A-4. 파티클 필터 (PF)
- **정의**: 비선형 + **비가우시안·다봉** 분포까지 다루는 몬테카를로 추정기.
- **동작원리**: 가능한 상태를 수많은 **파티클(가중 샘플)** 로 표현 → 예측 → 가중치 갱신 → 리샘플링 반복.
- **사용용도**: 로봇 전역 위치추정(납치 문제), 다봉 추적, 영상 객체 추적. (비용 큼)

### A-5. 정보 필터 (Information Filter)
- **정의**: KF를 공분산 `P` 대신 **정보행렬 `P⁻¹`** 로 표현한 쌍대(dual) 형태.
- **동작원리**: 보정이 단순 덧셈이 되어 다수 센서·노드 융합이 쉬움.
- **사용용도**: 분산 센서 네트워크, 멀티 로봇 협업 SLAM.

---

## Part B. 평활기 (Smoother)

### B-1. RTS 평활기 (Rauch–Tung–Striebel)
- **정의**: **미래 데이터까지** 활용해 과거 상태를 다시 추정하는 오프라인 추정기.
- **동작원리**: 정방향 KF 실행 후 **역방향으로 한 번 더** 보정(forward–backward).
- **사용용도**: 실시간 아닌 후처리(모션 캡처 정제, 궤적 재구성).

---

## Part C. 자세 추정용 경량 필터

### C-1. 보완 필터 (Complementary Filter)
- **정의**: 주파수 특성이 상보적인 센서를 합치는 단순 필터.
- **동작원리**: 한 센서는 저역통과(가속도계), 다른 센서는 고역통과(자이로)로 거른 뒤 가중합.
  ```
  θ = α·(θ + 자이로·dt) + (1−α)·가속도각
  ```
- **사용용도**: 드론·IMU 자세(roll/pitch). 가볍고 튜닝 쉬움 → 칼만의 실전 대안.

### C-2. Madgwick / Mahony 필터
- **정의**: 쿼터니언 기반 IMU/AHRS 자세 추정 표준.
- **동작원리**: 자이로 적분 자세를 가속도·지자기 측정으로 경사하강(Madgwick) / PI 보정(Mahony)하여 드리프트 제거.
- **사용용도**: 웨어러블, 드론 FC(ArduPilot 등), VR 트래커.

---

## Part D. 전통 신호처리 필터 (주파수 선택)

### D-1. 이동평균 / 지수가중평균 (MA / EMA)
- **정의**: 가장 단순한 저역통과 평활 필터.
- **동작원리**: MA는 최근 N개 평균, EMA는 `y = α·x + (1−α)·y_prev` 로 과거에 지수 감쇠 가중.
- **사용용도**: 빠른 노이즈 제거, 주가 추세선, 임베디드 경량 평활.

### D-2. Butterworth / Chebyshev / Bessel (IIR)
- **정의**: 특정 대역을 통과/차단하는 디지털·아날로그 필터 설계군.
- **동작원리**: 전달함수 극점 배치로 특성 결정 — Butterworth(평탄), Chebyshev(가파름+리플), Bessel(선형 위상).
- **사용용도**: 오디오, 생체신호(EEG/ECG), 안티앨리어싱.

### D-3. FIR 필터
- **정의**: 유한 임펄스 응답, 피드백 없는 항상 안정한 필터.
- **동작원리**: 입력과 고정 계수의 합성곱(convolution). 선형 위상 구현 가능.
- **사용용도**: 위상 왜곡이 치명적인 통신·오디오.

### D-4. Notch 필터
- **정의**: 특정 단일 주파수만 제거하는 대역 차단 필터.
- **동작원리**: 목표 주파수에 깊은 골(notch)을 형성.
- **사용용도**: 60/50Hz 전원 험 제거, 생체신호 노이즈 제거.

### D-5. Savitzky–Golay
- **정의**: 형태(피크·폭)를 보존하는 평활 필터.
- **동작원리**: 슬라이딩 윈도우 안에서 다항식을 최소제곱 피팅해 중앙값을 대체.
- **사용용도**: 스펙트럼 분석, 피크가 중요한 과학 데이터.

---

## Part E. 비선형 / 통계 필터

### E-1. 중앙값 필터 (Median Filter)
- **정의**: 윈도우 내 중앙값으로 대체하는 비선형 필터.
- **동작원리**: 평균 대신 정렬 후 중앙값 → 튀는 값(impulse)에 강건, 엣지 보존.
- **사용용도**: 소금-후추 노이즈 제거, 센서 outlier 제거, 영상 전처리.

### E-2. 양방향 필터 (Bilateral Filter)
- **정의**: 엣지를 보존하는 영상 평활 필터.
- **동작원리**: 거리(공간) + 화소값 차이(밝기) 두 가중치를 곱해 비슷한 화소만 평균.
- **사용용도**: 사진 노이즈 제거, 윤곽 유지 보정.

### E-3. 위너 필터 (Wiener Filter)
- **정의**: 평균제곱오차(MSE)를 최소화하는 최적 선형 복원 필터.
- **동작원리**: 신호·노이즈 전력 스펙트럼 비율로 주파수별 최적 이득 결정.
- **사용용도**: 영상 디블러링, 음성 노이즈 제거.

---

## Part F. 적응 필터 (Adaptive)

### F-1. LMS / RLS
- **정의**: 환경에 맞춰 계수를 스스로 갱신하는 필터.
- **동작원리**: 오차를 줄이는 방향으로 계수 업데이트 — LMS는 경사하강(느림·안정), RLS는 최소제곱(빠름·무거움).
- **사용용도**: 에코 캔슬링, 능동 소음 제거(ANC), 채널 등화.

---

## Part G. 선택 가이드 (가져가실 한 장)

```
동역학 모델 있음 + 선형        → KF
동역학 모델 있음 + 비선형       → EKF → (자코비안 어려우면) UKF
비가우시안·다봉(전역 위치추정)  → Particle Filter
IMU 자세만, 가볍게            → Complementary / Madgwick
단순 노이즈 평활              → EMA / Moving Average
피크 보존 평활               → Savitzky–Golay
튀는 값(outlier) 제거         → Median
특정 주파수 제거             → Notch / Butterworth
에코·소음 제거               → LMS / RLS
실시간 아님(후처리 정밀)       → RTS Smoother
```

| 상황 | 추천 |
|------|------|
| 선형 모델 + 센서융합 | KF |
| 비선형 모델 | EKF → UKF |
| 비가우시안·다봉 | Particle |
| IMU 자세(경량) | Complementary / Madgwick |
| 단순 평활 | EMA / MA |
| 피크 보존 | Savitzky–Golay |
| outlier 제거 | Median |
| 주파수 제거 | Notch / Butterworth |
| 에코·소음 | LMS / RLS |

## 한 줄 요약

> **"동역학 모델이 있나?"** 가 첫 갈림길이다. 있으면 칼만 계열(선형 KF → 비선형 EKF/UKF → 비가우시안 PF)로 *상태*를 추정하고, 없으면 신호처리 필터로 *주파수*를 거른다. 칼만이 항상 정답은 아니며 — IMU 자세처럼 가벼운 곳엔 보완 필터, 튀는 값엔 중앙값 필터처럼 **문제에 맞는 가장 단순한 필터가 정석**이다.
