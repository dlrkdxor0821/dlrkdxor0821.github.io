NCNN 학습 노트

온디바이스 추론 엔진 NCNN 개념 정리. AI 모델의 학습/추론 단계 구분, NCNN의 위치(ONNX·TFLite와 같은 급의 대체 런타임), 빠르고 가벼운 이유, 모델 파일 구조, 실제 배포 흐름, YOLO 예시, 한계까지.

먼저 큰 그림부터

AI 모델은 보통 두 단계를 거친다.

  1. 학습(Training) — 데이터로 모델을 똑똑하게 만드는 과정. 무겁고 GPU 필요. → PyTorch 영역
  2. 추론(Inference) — 다 만든 모델로 실제 예측하는 과정. → 배포 단계

문제는, 학습은 좋은 PC에서 하지만 실제 서비스는 폰이나 라즈베리파이처럼 힘 약한 기기에서 돌아가야 할 때가 많다는 것. PyTorch를 그대로 폰에 올리면 너무 무겁고 느리다.

그래서 추론만 가볍게 전담하는 도구가 따로 필요한데, 그게 NCNN이다.

NCNN이 정확히 뭐냐

Tencent가 만든 C++ 기반 추론 전용 엔진. 처음부터 목표가 딱 하나 — "ARM(폰·라즈베리파이) 같은 작은 기기에서 모델을 최대한 빠르고 가볍게 돌린다."

핵심은 추론만 한다는 것. 학습 기능은 아예 없다. "다 만든 모델을 실행하는 플레이어"라고 보면 된다.

어디에 위치하냐 (이게 중요)

NCNN은 TFLite나 ONNX Runtime을 빠르게 해주는 게 아니라, 그것들과 같은 자리에서 경쟁하는 대체재다.

PyTorch 로 학습
   ↓ export
ONNX (프레임워크 간 공용 교환 포맷)
   ↓ 여기서 배포 런타임 선택
   ├─ ONNX Runtime   (범용, 서버·데스크톱)
   ├─ TFLite         (구글·안드로이드 생태계)
   └─ NCNN           (ARM·모바일 극한 최적화)   ← 같은 급

즉 "ONNX/TFLite 위에 NCNN"이 아니라 "ONNX/TFLite 대신 NCNN"이다.

왜 빠르고 가벼운가 (NCNN의 자랑 3가지)

  • 의존성이 없다 — BLAS, protobuf 같은 무거운 외부 라이브러리를 안 쓴다. 그래서 빌드 결과물이 작고 어디든 잘 들어간다
  • ARM에 손최적화 — ARM 칩의 가속 명령(NEON)에 맞춰 연산을 직접 깎아서 CPU만으로도 빠르다
  • GPU도 지원 — Vulkan으로 폰 GPU나 라즈베리파이 GPU 가속도 가능 (CUDA 없이도)

모델은 어떻게 생겼나

NCNN 모델은 항상 파일 2개 한 쌍이다.

파일 내용
.param 네트워크 구조 (어떤 레이어가 어떻게 연결됐는지)
.bin 가중치 (학습으로 얻은 숫자값)

다른 프레임워크 모델을 이 두 파일로 변환해서 사용한다.

실제 흐름 한 번에

PyTorch 학습
  → ONNX 로 export
  → NCNN 포맷(.param/.bin)으로 변환
  → (선택) 양자화로 더 작고 빠르게  ※ 정밀도 살짝 낮춰 속도↑
  → 기기에 올려서 추론 실행

대표 예시

PC(GPU)에서 학습한 YOLO 객체탐지 모델을 NCNN으로 변환 → 라즈베리파이에서 실시간으로 카메라 영상 속 물체를 탐지. GPU 없는 작은 기기에서 YOLO 돌릴 때 정말 흔한 조합이다.

한계도 솔직히

  • 추론 전용 (학습 불가)
  • 변환할 때 NCNN이 지원 안 하는 연산이 있으면 직접 손봐야 함
  • 문서가 영어/중국어 위주라 처음엔 좀 헤맬 수 있음

핵심 요약

PyTorch(연구) → ONNX(교환) → NCNN(ARM 기기에서 빠르게 실행). 작은 기기용 경량 추론 엔진.

TFLite vs NCNN vs MNN

한눈 비교

만든 곳 가장 강한 환경 한마디
TFLite (현 LiteRT) Google 안드로이드 + 하드웨어 가속(NPU/DSP) 생태계·툴링 끝판왕
NCNN Tencent 순수 ARM CPU + 임베디드/CV 가볍고 의존성 0
MNN Alibaba 다양한 백엔드 균형(CPU/GPU 골고루) 범용 올라운더

실전 선택 가이드 (시나리오별)

상황 추천
안드로이드 앱 + NPU/Edge TPU 쓰고 싶다 TFLite
라즈베리파이에서 YOLO 실시간 NCNN
비전 모델, 최소 용량으로 임베디드 NCNN
iOS·안드로이드·엣지를 한 엔진으로 통일 MNN
GPU 가속까지 골고루 챙기고 싶다 MNN (또는 TFLite GPU delegate)
처음 배우는 중, 자료 많은 게 최고 TFLite
